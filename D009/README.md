# 동해 수온과 참다랑어류 어획량 분석 보고서

## 1. 데이터 개요
- 출처: FAO Global Capture Production (한국, 참치류) + 국립수산과학원 실시간 해양수산환경 관측시스템
- 기간: 1950~2024년 (FAO 어획량), 2011~2025년 (동해 수온 관측)
- 원본 행 수 / 정제 후 행 수: 5,400행 (FAO melt 직후) → 375행 (한국 근해 필터링 후) → 70행 (수온 데이터와 병합 후)

## 2. 발견한 품질 문제
- 결측: FAO 데이터 melt 직후 catch_amount에 75개(1.4%) 결측 발견 → 확인 결과 실제 결측이 아니라 원본 엑셀의 출처 안내·합계 각주 행이 섞여 들어온 것이었음
- 중복: FAO 원본 데이터에 완전 중복 행 0건 (문제 없음)
- 이상치: 뚜렷한 수치 이상치는 발견되지 않았으나, 특정 어종(Bigeye, Pacific bluefin, Skipjack, Yellowfin tuna)이 한국 근해 해역에서 전 기간(2011~2024) 어획량 0으로 기록되는 특이 패턴 발견
- 표기 혼재: FAO 데이터는 가로형(wide) 구조로 연도가 컬럼으로 펼쳐져 있어, 분석을 위해 세로형(long)으로 변환(melt) 필요

## 3. 처리 결정과 근거 (5줄 이상)
1. 중복 행: FAO 원본에 완전 중복 0건 확인 → 별도 처리 불필요
2. 각주/합계 행 제거: ASFIS species(어종명) 컬럼이 결측인 행 225개 제거 → 원본 엑셀 하단의 출처·합계 각주가 melt 과정에서 데이터 행으로 오인되었기 때문
3. 조업해역 한정: FAO 조업해역 중 `Pacific, Northwest`만 필터링 → 참치는 원양성 어종으로 대부분 대서양·인도양에서 잡히며, 한국 근해(동해 포함)와 가장 관련 깊은 해역이 이곳이기 때문. 단, 이 해역은 동해뿐 아니라 서해·일본 근해도 포함하는 광역 구분임
4. 수온 데이터 결측치 처리: 178개 월별 csv(326만 행) 중 표층수온(℃) 결측 행 제거 후, 연도별 평균으로 집계 → 원본 데이터가 30분 간격 관측치라 결측이 일부 존재했으며, FAO 연 단위 데이터와 결합하려면 연평균으로 압축이 필요했음
5. 데이터 병합 방식: 연도(year) 기준 inner join 사용 → 양쪽 데이터가 모두 존재하는 2011~2024년(14년)만 비교 대상으로 삼기 위함
6. 저장 형식: Parquet과 CSV를 함께 저장 → Parquet은 자료형 보존과 압축 효율이 높아 분석용으로, CSV는 범용 호환성 확인용으로 병행 저장

## 4. 주요 KPI 결과
- (월·매장 관점 → 어종·연도 관점) 2018년에 `Tuna-like fishes NEI`(미분류 참치류)의 어획량이 77,845톤으로 최고치를 기록했으며, 이는 2024년(수온 최고치 17.36℃) 대비 오히려 높은 수치임
- (상품 관점 → 어종 관점) 5개 어종 중 `Tuna-like fishes NEI`만 유의미한 어획량(총 306,588톤)을 보였고, 나머지 4개 세부 어종은 이 해역에서 전 기간 0톤으로 집계되어 세부 어종 통계보다 미분류 범주 의존도가 높음을 확인함
- 수온-어획량 상관계수: 0.277 (약한 양의 상관관계)

## 5. 한계와 후속 작업
- FAO 데이터는 국제 통계 특성상 1~2년의 보고·검증 시차가 있어 2024년까지만 제공되며, 2025~2026년 최신 동향은 이 데이터만으로 판단할 수 없음
- `Pacific, Northwest` 해역은 동해뿐 아니라 서해·일본 근해까지 포괄하는 광역 구분이라, 동해에 국한된 정밀한 분석에는 한계가 있음
- 세부 어종별 어획량이 대부분 0으로 기록되어 있어, 실질적으로는 `Tuna-like fishes NEI` 범주에만 의존해 분석함 — 향후 국내 통계(국립수산과학원 어업생산동향조사 등)로 세부 어종 구분을 보완할 필요가 있음
- 수온-어획량 상관계수(0.277)가 약한 수준으로, 조업 노력·국제 어업 규제(TAC)·어군 이동 경로 등 다른 요인을 함께 고려해야 더 정확한 해석이 가능함
- 후속 분석에서는 국립수산과학원의 국내 어업생산동향조사 2025년 잠정치를 추가로 활용해 최신 동향을 보완할 수 있음
